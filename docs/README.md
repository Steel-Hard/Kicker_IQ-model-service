# Docs

Este diretorio contem a documentacao funcional das rotas e contratos do microservico.

## Arquivos

- routes/README.md: indice das rotas disponiveis.
- routes/match.md: endpoints de predicao individual e em lote.
- routes/athlete.md: endpoints de analise por atleta.
- routes/meta.md: metadados de classes.
- routes/health.md: saude de banco/modelo/scaler.
- routes/report.md: exportacoes JSON e CSV.

## Escopo

- Foco em novas rotas para ajudar frontend e backend.
- Mudancas de metricas/classes do modelo podem ser documentadas em novas paginas quando iniciadas.
